# Space Invaders

A classic Space Invaders game implemented in Python using the Pygame library.

## Features
- Dynamic shooting system
- Scoring system
- Various types of enemies
- A boss with unique attack patterns
- Dash system
- Sound and visual effects

## Requirements
- Python 3.x
- Pygame

## Installation
1. Clone the repository
2. Install the required libraries: `pip install pygame`
3. Run the game: `python SpaceInvadersGame.py`

## Controls
- Arrow keys: Movement
- Spacebar: Shoot
- Shift: Dash
